# 🏫 안심 캠퍼스 맵 (Safe Campus Map)
> **대학 캠퍼스 내 분실물·습득물의 실시간 위치 매핑, 1:1 안전 채팅 및 최적 도보 동선 안내 서비스**

안심 캠퍼스 맵은 대학교 구성원(학생, 교직원)들이 캠퍼스 내에서 유실물을 실시간 지도 위 핀(Pin)과 1:1 채팅을 통해 신속하게 찾고, 매칭 시 수령지까지의 안전한 도보 동선 안내를 제공하는 통합 서비스입니다.

---

## 🌟 핵심 기능 (Core Features)

* **🗺️ 실시간 지도 위 핀 매핑 (Interactive Pinning)**
  * 분실물(🔴 Red)과 습득물(🔵 Blue)의 직관적인 지도 핀 시각화
  * 핀 위치의 도로명 주소 및 주변 건물명 자동 변환 (Reverse Geocoding)
* **📡 고정밀 실시간 GPS 연동 및 PC/모바일 라이브 모션 트래킹**
* **🤝 1:1 인앱 안전 채팅 & 약속 장소 조율**
  * 개인정보(연락처, 카톡 ID) 노출 없는 안전 소통 창구
  * 대화 도중 만나거나 물건을 전달할 약속 장소(좌표) 조율 기능
* **🧠 유사 물품 스마트 매칭 및 자동 이메일 알림**
  * 카테고리/거리(500m 이내)/제목 유사성 분석을 통한 실시간 매칭 추천
  * 매칭 발생 시 실시간 인앱 토스트 팝업 및 이메일 자동 발송
* **🔒 대학 웹메일 OTP 본인 인증 (Auth Shield)**
  * `*.ac.kr` / `*.edu` 메일 전용 OTP 보안 가입 절차
  * 가입 계정에 기반한 대학별 커스텀 캠퍼스 맵 테마 자동화 세팅
* **🚸 카카오 모빌리티 API 연동 실시간 도보 동선 안내 (Navigation)**
  * 사용자 출발지부터 습득물(또는 약속 장소) 수령지까지의 안전 최단 도보 경로 (`Polyline` 렌더링)
  * 실시간 남은 거리, 소요시간(ETA) 및 도착 감지(15m 이내) 모션 피드백
* **🌐 다국어 완벽 지원 (Multilingual UI)**
  * 한국어(KO), 영어(EN), 베트남어(VI) 실시간 원클릭 언어팩 전환

---

## 🛠️ 기술 스택 (Tech Stack)

* **Frontend**: Next.js 14+ (App Router), TypeScript, Vanilla CSS (Sleek Glassmorphism UI)
* **Backend & DB**: Supabase (PostgreSQL DB, Realtime Message Sync, Storage Bucket)
* **APIs**: Kakao Maps JS SDK, Kakao Mobility Directions REST API, Nodemailer SMTP

---

## 🦺 안심 캠퍼스 이용 수칙 (Safety First)

1. **공개 대면 장소 권장**: 대면 인계 시 중앙도서관 입구, 학생회관 로비 등 사람이 많고 CCTV가 있는 교내 공개 장소를 활용해 주세요.
2. **비대면 보관소 적극 활용**: 학과 행정실이나 무인 보관함에 물품을 맡기고 핀 좌표를 전송하여 전달하시는 것을 추천합니다.
3. **상태 사진 보관**: 습득 당시의 스크래치나 파손 부위를 미리 여러 장 촬영해 보관하시면 오해나 분쟁을 예방할 수 있습니다.
